# Train Tracker

Train Tracker is a Pakistan Railways schedule explorer built with React and Vite.

It uses timetable and station summary information curated from the public Pakistan Railways / PakRail website on April 29, 2026. The app is intentionally positioned as a schedule explorer, not a live GPS tracker, because no public official realtime operations feed is exposed for this route set.

## Scripts

- `npm install`
- `npm run dev`
- `npm run test`
- `npm run build`

## Data source

- `https://pakrail.pk/`

## Deploy

Netlify build settings:

- Build command: `npm run build`
- Publish directory: `dist`
